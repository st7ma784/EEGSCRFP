📊 6 PATHWAY METRICS
────────────────────

1. Routing Sparsity          → Effective number of active paths
2. Path Competition Index    → max(attn) / mean(attn) [winner-take-all]
3. Path Efficiency           → Energy in top-20% weights
4. Routing Entropy           → Shannon entropy of attention
5. Inter-head Divergence     → KL divergence between heads
6. Layer-wise Stability      → Cosine similarity across layers

⚙️  SYSTEM ARCHITECTURE
────────────────────

Input: Text prompt + Sparsity level
  ↓
SparseAttentionWrapper
  • Load HF transformer (GPT-2 or similar)
  • Apply top-k attention masking (dynamic sparsity)
  ↓
PathwayMetricsComputer
  • Extract 6 routing metrics [6D vector]
  ↓
EEGProjector
  • Linear projection 6D → 105D [EEG-like signal]
  • Add Gaussian noise (σ=0.1)
  ↓
Predictors (2 parallel MLPs)
  • PathwayPredictor: 6D → sparsity (r = 0.68)
  • EEGPredictor: 105D → sparsity (r = 0.42)
  ↓
Loss = MSE(pathway_pred, sparsity) + MSE(eeg_pred, sparsity)
