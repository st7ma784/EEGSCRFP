📦 CORE COMPONENTS (FULLY IMPLEMENTED)
───────────────────────────────────────

1. SPARSE ATTENTION MODEL
   Location: src/model/sparse_attention.py
   Features:
   - Wraps HuggingFace transformers
   - Top-K attention masking (dynamic sparsity control)
   - Extracts attention maps at each sparsity level
   - Freezes base model, uses inference-only sparsity

2. PATHWAY METRICS (6 Measures)
   Location: src/metrics/pathway_metrics.py
   Metrics:
   ✓ Routing Sparsity (effective number of active paths)
   ✓ Path Competition Index (winner-take-all strength)
   ✓ Path Efficiency (energy in top-k weights)
   ✓ Routing Entropy (attention distribution entropy)
   ✓ Inter-head Divergence (KL divergence between heads)
   ✓ Layer-wise Stability (cosine similarity across layers)

3. EEG PROJECTION
   Location: src/projection/eeg_projector.py
   Features:
   - Linear projection: 6 pathway features → 105 EEG channels
   - Gaussian noise injection (σ = 0.1)
   - Optional temporal smoothing
   - Learned via gradient descent

4. MLP PREDICTORS (2 Independent)
   Location: src/predictor/mlp.py
   Architecture:
   - Input → ReLU(256) → ReLU(128) → Output
   - Input dim 1: pathway_encoder (6D)
   - Input dim 2: eeg_encoder (105D)
   - Output: sparsity level prediction

5. DATASET & TOKENIZATION
   Location: src/data/dataset.py, tokenizer.py
   Features:
   - Fixed set of 10 prompts (content invariant)
   - 5 sparsity levels (0.1 - 0.9)
   - 5 samples per condition = 500 total samples
   - 80/20 train-val split

6. LIGHTNING MODULE
   Location: src/lightning_module.py
   Orchestrates:
   - Forward pass through entire pipeline
   - Training step with dual loss
   - Validation with correlation logging
   - Model checkpointing

7. EXPERIMENT RUNNERS (3 Experiments)
   Location: experiments/runner.py
   Experiment 1: Pathway → Sparsity prediction
   Experiment 2: EEG → Sparsity prediction
   Experiment 3: Pathway → EEG transfer learning

8. CONFIGURATION SYSTEM
   Location: config/config.py
   Features:
   - Dataclass-based configuration
   - Comprehensive parameter management
   - Easy overriding via CLI
   - Default configs provided

🔧 KEY PARAMETERS
─────────────────

Model:
  - Base model: GPT-2 (124M params, easily swappable)
  - Sparsity levels: [0.1, 0.3, 0.5, 0.7, 0.9]
  - Sparsity type: Top-K masking

Metrics:
  - Extracted: 6 pathway metrics (Rényi entropy, competition index, etc.)
  - Normalized: Z-score standardization

Projection:
  - Input: 6 pathway features
  - Output: 105 EEG channels
  - Transformation: Linear (6×105 matrix)
  - Noise: Gaussian σ=0.1

Training:
  - Optimizer: AdamW (lr=1e-3, weight_decay=1e-5)
  - Loss: MSE(pathway_pred, sparsity) + MSE(eeg_pred, sparsity)
  - Epochs: 20
  - Batch size: 8
  - Val split: 80/20
